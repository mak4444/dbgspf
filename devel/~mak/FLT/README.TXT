 Библиотека создана на базе библиотеки Якимова Д. А.
 из пакета DEY для SPF370 ( http://www.forth.org.ru/~day/day.zip ).
 Его библиотека быстрая и точная, но используемый им
 аппаратный стек ограничен 8-ю элементами.
 Я создал 2-а уровня для SPFOpt. Слова использующие для
передачи чисел с плавающей точкой стек сопроцессора помечены
суффиксом '_', в противном случае используются стек данных
Форта. Слова использующие стек данных Форта могут
использовать слова использующие для передачи стек 
сопроцессора. Переключение режима производится
путем перемещения данных: 
T>F - из  стека данных встек сопроцессора
F>T - обратня T>F
